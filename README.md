1️⃣ What is the difference between var, let, and const?
Answer: var,let and const are type of variables in JavaScript.var is function-scoped and can be redeclared and let is block-scoped and cannot be redeclared in the same block.const is also block-scoped but its value cannot be changed after it is assigned.Its contains constant value always.

2️⃣ What is the spread operator (...)?
Answer: The spread operator is used to expand elements of an array or object. It helps to joint arrays and objects easily.For example, [...ar1, ...ar2] combaine two arrays into one.

3️⃣ What is the difference between map(), filter(), and forEach()?
Answer: map() creates a new array by changing each element of the original array.
Filter() creates a new array with elements that fullfilled given condition.

forEach() only runs a function for each element and does not return a new array.

4️⃣ What is an arrow function?
Answer: An arrow function is a short way to write a function in JavaScript using =>.It makes the code smaller and easier to read.Example: const multiply = (a, b) => a * b.

5️⃣ What are template literals?
Answer:Template literals are strings written with backticks ` ` in JavaScript. They allow variables to be added inside the string  by using ${}.Its also make easier to write multi-line strings.
